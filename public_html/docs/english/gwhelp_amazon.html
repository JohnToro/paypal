Gateway Configuration
Amazon SimplePay

Option Description
Production Access Key Enter the Access Key assigned to your Production Amazon seller account.
Production Secret Key Enter the Secret Key assigned to your Production Amazon seller account.
Test (Sandbox) Access Key Enter the Access Key assigned to your Testing Amazon seller account.
Test (Sandbox) Secret Key Enter the Secret Key assigned to your Testing Amazon seller account.
Amazon URL- Production Enter the production Amazon URL, e.g. "https://authorize.payments.amazon.com"
Amazon URL- Sandbox Enter the Amazon Sandbox URL, e.g. i"https://authorize.payments-sandbox.amazon.com"
Testing (Sandbox) Mode? Check this box during testing. The Sandbox URL will be used instead of the Production URL.
Enabled Check this box to indicate that this gateway is enabled for use. If not checked, the PayPal gateway will not be used.
Buttons Select the buttons that are supported by this gateway. Payment for a shopping cart is always supported. This is primarily meant to reduce clutter in blocks and product lists, where you may want only one gateway handling immediate purchases.

Note that to use "Donation" buttons, you must be set up as a valid non-profit organization with Amazon.
Order Enter a number to represent the priority of this gateway (lower number means higher priority).
